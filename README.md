## ✨ Features

- **Dark, modern design** — black and orange aesthetic with custom typography
- **Smooth animations** — staggered hero load, scroll-triggered reveals, hover effects
- **Fully responsive** — mobile, tablet, and desktop ready
- **Auto-deployed** — every push to `main` deploys instantly via Vercel
- **Fast** — built with Vite for near-instant load times
- **Accessible** — semantic HTML, valid href attributes, proper contrast

## 🛠 Tech Stack

- **Frontend** — React 18, Vite, CSS Modules
- **Fonts** — Bebas Neue, Syne, DM Sans, JetBrains Mono
- **Deployment** — Vercel (auto-deploy from GitHub `main`)





## 📦 Build

```bash
npm run build      # production build → /dist
npm run preview    # preview production build locally
```

---

## 🌍 Deployment

Deployed on **Vercel** with automatic CI/CD.

- Framework preset: **Vite**
- Build command: `npm run build`
- Output directory: `dist`
- Branch: `main` → auto-deploys on every push
